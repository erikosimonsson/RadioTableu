# RadioTableu
Tableu app for radio channels under Sveriges Radio (Swedish public service radio). Frontend made with the Java Swing library, tableu data obtained from SR's public API.

Launch the app safely by downloading the `.java`-files and compiling them into a `.jar` file. Open a command line tool, navigate to the folder containing your `.jar`-file and launch it with `java -jar nameOfYourFile.jar`.

If you are trusting, you can instead download the radioinfo.jar file. Open a command prompt/terminal, navigate to the folder containing radioinfo.jar and run it with `java -jar radioinfo.jar`.

Please note that the app was made in Swedish for a university course.
